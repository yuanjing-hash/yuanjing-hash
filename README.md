**[OhMyCine](https://github.com/yuanjing-hash/OhMyCine)** is an open-source, cross-platform, self-hosted home cinema system.

It is currently focused on the standalone Player experience: a Tauri + Vue desktop app with libmpv playback, Emby/OpenList/Alist data sources, local read-only scraping, TMDB metadata enrichment, and a Cinema OS style interface.
